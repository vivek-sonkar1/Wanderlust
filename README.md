# üåç Wanderlust

Wanderlust is a full-stack travel listing web application where users can explore travel destinations, create their own listings, upload images, post reviews, and view listing locations on an interactive map.  
It is built using **Node.js**, **Express**, **MongoDB**, and integrates **Cloudinary** for image uploads and **Mapbox** for geolocation and mapping, and **Bootstrap** for UI styling.

---

## ‚ú® Features

- **User Authentication**
  - Register, Login, Logout using Passport.js
- **Listing Management**
  - Create, edit, delete, and view travel listings
- **Image Uploads**
  - Upload images using Multer and Cloudinary
- **Map Integration**
  - Interactive Mapbox map for location display
- **Review System**
  - Leave ratings & comments on listings
- **Server-Side Validation**
  - Joi input validation to prevent invalid data
- **Client-Side Validation**
  - HTML5 form validation + Bootstrap validation styles for user-friendly input feedback
- **Session and Flash Messages**
  - Display success/error alerts throughout the application
